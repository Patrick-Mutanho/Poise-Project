# Project Management Tool

## Project Overview 
This project is a project management tool for a structural engineering company developed in java.The tool allows the company to capture financial data, 
subcontractor data and project timelines.

## Project Functionality
1. The user can add project details such as start dates, deadlineline, and financial details of the project.
2. Furthermore, information such as contractors and architects associated to the project can be added be cpatured.
3. All the information added for each project can be changed as the project progress or if need be.
4. All the projects can be accessed and displayed along with information such as deadlines and outstanding amounts.

## Project Aim
The aim of the project is to assist a company manage projects and aquire and maitain critical data for each project.
